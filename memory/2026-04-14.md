# 2026-04-14

## Scope

Initial planning and repo scaffolding for `WPA3-Enterprise` Wi-Fi with `EAP-TLS` on `rainier`.

## Key decisions

- UniFi will remain the Wi-Fi infrastructure and `802.1X` authenticator layer.
- FreeRADIUS on `rainier` is the chosen authentication server.
- An internal CA is required for the RADIUS server certificate and Apple client certificates.
- The practical first rollout will use Docker for FreeRADIUS on `rainier`.
- `step-ca` is the chosen CA service for the first live PKI implementation.

## Repo changes

- Added `docs/RADIUS-WPA3-EAP-TLS.md` to capture the target architecture and rollout phases.
- Added `compose/freeradius/` as the initial Docker scaffold.
- Added `compose/step-ca/` as the initial internal CA scaffold.
- Added firewall and services documentation for planned RADIUS exposure and runtime paths.
